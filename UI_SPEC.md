# UI 风格收敛说明

参考图像体现的是一种深色三栏工作台：

- 左栏：会话 / Agent / 导航
- 中栏：聊天时间线
- 右栏：详情区、文件、统计、说明

用于本项目时，建议映射为：

## 左栏
- Debate 列表
- Topic 列表
- 当前参与 Agent
- 快捷操作按钮

## 中栏
- 当前 Session 标题
- 中间滚动聊天区
- 每个 turn 以气泡 + 摘要显示
- 底部放控制区：开始 / 暂停 / 恢复 / 中断

## 右栏
- 当前争议点
- claims / attacks / concessions
- judge hints
- winner card

## 视觉建议
- 背景用深蓝灰，不要纯黑
- 卡片圆角偏大
- 文字层级清晰，说明文字弱化
- 气泡不要太扁平，保留轻微面板感
